# 🎮 SonicCyclops 2D Game Project

A 2D game built with **Godot**, featuring music composed with **LilyPond** and sprites created with **LibreSprite**.

## 🛠️ Toolchain

- **Game Engine:** [Godot](https://godotengine.org/) - Open source 2D/3D game engine
- **Music Composition:** [LilyPond](https://lilypond.org/) - Text-based music notation system
- **Sprite Art:** [LibreSprite](https://libresprite.github.io/) - Pixel art editor (Aseprite fork)

## 📁 Project Structure

```
├── src/           # Godot project files and scripts
├── assets/        # Game assets
│   ├── sprites/   # LibreSprite files and exported PNG/animations
│   ├── music/     # LilyPond files and exported audio
│   └── audio/     # Sound effects and other audio
├── docs/          # Game design documents
└── tools/         # Build scripts and asset pipelines
```

## 🚀 Development Workflow

1. **Game Logic:** Develop in Godot with GDScript/C#
2. **Music Creation:** Compose with LilyPond text notation → export to audio
3. **Sprite Creation:** Design in LibreSprite → export spritesheets/animations
4. **Integration:** Import assets into Godot project
